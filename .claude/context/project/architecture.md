# Architecture — Clean & Modular (Simdikta)\n+\n+Last updated: 2026-03-23\n+\n+## Layering (Clean Architecture)\n+- **Domain**: Entities, Value Objects, domain services, domain errors. No external imports.\n+- **Service/Application**: Orchestrates use cases, owns transactions, depends only on domain interfaces.\n+- **Handler/Interface**: HTTP decode → call service → encode. No business logic; never call repositories directly.\n+- **Repository (postgres)**: Implements domain repository interfaces; SQL only; always scoped by `school_id`.\n+- **Adapter**: External systems (selection engines, payment, WhatsApp, queue, maps). Implements domain interfaces.\n+- **pkg**: Internal shared utilities (crypto, masking, response, pagination) — avoid business logic here.\n+\n+Dependency rule: imports point inward (handler → service → domain). Violations are blockers.\n+\n+## Bounded Contexts\n+- Registration (pendaftaran)\n+- Verification (verifikasi dokumen)\n+- Selection (zonasi/domilisi, prestasi, afirmasi, mutasi)\n+- Payment (midtrans/xendit)\n+- Notification (WhatsApp/SMS)\n+\n+## Multi-tenant contract\n+- Every repository function signature: `func X(ctx context.Context, schoolID uuid.UUID, ...)`.\n+- Every SQL query on multi-tenant tables starts with `WHERE school_id = $1`.\n+- No cross-tenant joins or aggregations.\n+\n+## Audit & status changes\n+- Table `ppdb_audit_logs` is INSERT-only.\n+- Any change to `ppdb_applicants.status` must INSERT audit log **within the same transaction, before** the status update.\n+\n+## Selection engine (risk-first)\n+- Riskiest assumption: correctness of selection engine (Haversine + formula). Build and test it before UI.\n+\n+## Data flow overview\n+- HTTP handler → service/use case → domain entities/VO → repository (PostgreSQL) and adapters (payments, notifications, queue).\n+- Redis: OTP (5m TTL), rate limiting (sliding window), queue (BLPOP), public dashboard cache (60s).\n+\n+## Deployment basics\n+- Local: Docker Compose. Production: Kubernetes/EKS + Terraform. CI: GitHub Actions.\n*** End Patch ***!
